# Calculadora de IMC

Este projeto é uma simples calculadora de IMC (Índice de Massa Corporal), que permite ao usuário inserir seu peso e altura para calcular seu IMC e visualizar a classificação de acordo com os padrões da Organização Mundial da Saúde (OMS).

## 📌 Tecnologias Utilizadas
- HTML
- CSS
- JavaScript

## 📋 Funcionalidades
- Entrada de peso e altura.
- Cálculo do IMC.
- Exibição da classificação do IMC.
- Botão para limpar os campos de entrada e o resultado.

## 🚀 Como Usar
1. Abra o arquivo `index.html` em um navegador.
2. Insira seu peso e altura nos campos correspondentes.
3. Clique no botão "Calcular" para ver seu IMC e a classificação correspondente.
4. Para limpar os campos e os resultados, clique no botão "Limpar".
